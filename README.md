## Install
  
for Ubuntu  
1. pipのインストール  
```bash
$ sudo apt install python3-pip
```
2. Mysqlclientのインストール  
```bash
$ sudo apt-get install python-pip python-dev libmysqlclient-dev
```

for Centos7
1. python3-develのインストール
```bash
$ sudo yum install python3-devel
```
2. Mysqlclientのインストール
```bash
$ sudo pip3 install mysqlclient
```
